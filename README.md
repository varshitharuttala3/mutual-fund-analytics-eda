# 📊 Mutual Fund Analytics - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Indian Mutual Fund industry data from 2022 to 2026. The analysis explores fund performance, Assets Under Management (AUM), SIP trends, investor demographics, portfolio holdings, sector allocation, benchmark indices, and market correlations.

This project was developed as part of the Capstone Project under Bluestock Fintech.

---

## 🎯 Objectives

- Analyze daily NAV trends of mutual fund schemes.
- Study AUM growth across major fund houses.
- Examine SIP inflow trends and identify peak periods.
- Analyze category-wise fund inflows.
- Study investor demographics and investment behavior.
- Compare T30 and B30 city participation.
- Analyze mutual fund folio growth.
- Evaluate fund performance using returns and Sharpe ratios.
- Build NAV return correlation matrices.
- Examine sector allocation of equity funds.
- Compare benchmark index performance.

## 📈 Key Findings

- SBI Mutual Fund maintained market leadership.
- SIP inflows reached an all-time high in December 2025.
- Equity folios dominated investor participation.
- Investors aged 26–35 contributed the largest share of investments.
- Banking and IT sectors accounted for major portfolio allocations.
- Small-cap funds generated the highest long-term returns.
